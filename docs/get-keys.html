Get Your API Keys

Open each service, copy your keys, then deploy. Takes ~5 minutes.

1. Supabase Required

Sign up or log in, create a new project, then go to Project Settings → API.

supabase.com/dashboard
  • Project URL → NEXT_PUBLIC_SUPABASE_URL
  • anon / public key → NEXT_PUBLIC_SUPABASE_ANON_KEY
  • Settings → Database → Connection string (URI) → DATABASE_URL

2. Google AI Studio (Gemini) Required

Click "Get API Key" → Create key.

aistudio.google.com/apikey
  • API Key → GEMINI_API_KEY

3. USDA FoodData Central Optional

Sign up and request an API key. Improves nutrition accuracy.

fdc.nal.usda.gov/api-key-signup
  • API Key → USDA_API_KEY

4. Brave Search Optional

Sign up for the free tier (2,000 queries/mo). Helps with restaurant menu lookups.

brave.com/search/api
  • API Key → BRAVE_API_KEY
Popup blocked? Click the links above manually to open each service.
